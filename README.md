<h2>시작 가이드</h2>
<br/>
<h3>Requirements</h3>
<br/>

For building and running the application you need:
- Node.js 20.1.0
- Npm 9.6.4

<br/>
<h3>Installation</h3>

```
$ git clone https://github.com/NextTeamB/ONEHOUR
$ cd ONEHOUR
```
<br/>
<h4>Front end</h4>

```
$ cd ONEHOUR
$ npm install
$ npm run dev
```
